# micronaut-playground
Experimenting with Micronaut

## Modules

- Client: This module contains a Retrofit Client that can be pulled into any module to make HTTP calls
- micronaut-basic: A barebones micronaut app
- micronaut-http: Experimenting with making HTTP calls in a micronaut service
- micronaut-postgres: How can we interact with a postgres database with Micronaut
